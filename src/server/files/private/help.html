Connected Cameras

This section displays the IP, Port, and Location of all connected cameras. It will show "No data available" if no cameras are connected.

To add a new camera, setup and activate the camera client on a machine that is connected to the desired camera.

User Management

The "Admin" page allows you to manage user accounts:

  • Username: Displays the username of each account.
  • Access Level:
    • 0: Administrator
    • 1: Regular User
  • Actions: Remove users by clicking the "Remove" button next to each account.

To add a new user, fill out the "Add User" form on the Admin page with the desired username and password, and click "Submit".

Suspect Search and Blacklist

On the "Search Modes" page, you can search for suspects and manage the blacklist.

  • Suspect Search:
    • Enter the full name in the search form and upload relevant images of the suspect to identify the suspect.
    • If the suspect is found in the system, the map will automatically update and display the relevent location and time of his appearences.
    • Suspect statistics will appear when pressing the "Show Statistics" button bellow the map.
  • Blacklist:
    • Additions: Fill in the form to add a suspect to the blacklist.
    • Table: View the current blacklist with options to remove individuals.
    • Functionality: If a blacklisted person is identified, the system will alert the user(On any page in the Web Application) by opening a popup map that displays the appropriate coordinates.

Statistics

The "Show Statistics" button on the Search Modes page provides detailed statistics about the suspect's search result:

  • Basic Numbers:
    • Total Sightings
    • Unique Locations Count
    • Unique Dates Count
  • Most Frequent Analysis:
    • Date
    • Date Occurrences Count
    • Location
    • Location Occurrences Count
  • Average Time Spent Per Location: A table displays the average time spent at each location.
  • Graphs:
    • Dates Graph
    • Locations Graph
  • Control Buttons: Close the statistics window using the "Close Statistics" button.
